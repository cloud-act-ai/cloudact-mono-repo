Track your OpenAI API usage including GPT-4, GPT-3.5, and other models.

## What You'll See

- Daily and monthly costs
- Usage by model (GPT-4, GPT-3.5, etc.)
- Token consumption trends

## Setup Steps

### Step 1: Get Your API Key from OpenAI

1. Go to [platform.openai.com/api-keys](https://platform.openai.com/api-keys)
2. Click **Create new secret key**
3. Give it a name like "CloudAct"
4. **Copy the key immediately** - it won't be shown again!

<Callout type="warning">
  Make sure to copy your API key before closing the dialog. OpenAI only shows it once.
</Callout>

### Step 2: Add to CloudAct

1. In CloudAct, go to **Settings** → **Integrations**
2. Click **OpenAI**
3. Paste your API key
4. Click **Validate & Save**

### Step 3: Verify Connection

After saving:
- You'll see a **Connected** status
- Your data will start syncing automatically
- Check your dashboard in a few minutes

## Troubleshooting

### "Invalid API key"

- Make sure the key starts with `sk-` or `sk-proj-`
- Check that the key hasn't been revoked in OpenAI
- Verify you have billing enabled on your OpenAI account

### "No data showing"

- Data syncs once per day
- Check back tomorrow if you just connected
- Ensure you have recent API usage on OpenAI

## Disconnect

To remove the integration:

1. Go to **Settings** → **Integrations** → **OpenAI**
2. Click **Disconnect**
3. Confirm the removal

Your historical data will remain in CloudAct.
