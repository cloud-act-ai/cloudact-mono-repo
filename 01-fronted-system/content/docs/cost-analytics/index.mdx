Cost Analytics provides detailed insights into your organization's spending across cloud providers, GenAI services, and SaaS subscriptions.

## Accessing Cost Analytics

Navigate to **Cost Analytics** in the sidebar to access the analytics dashboard.

## Overview Dashboard

The Cost Analytics Overview shows:

### Total Spending

- Combined costs across all categories
- Period-over-period change percentage
- Trend indicators (up/down arrows)

### Cost Breakdown

| Category | Description |
|----------|-------------|
| **Cloud** | AWS, GCP, Azure, OCI infrastructure costs |
| **GenAI** | API usage for AI models (OpenAI, Anthropic, Google) |
| **Subscriptions** | SaaS tools and services |

### Charts and Visualizations

- **Stacked bar charts** - Daily/weekly cost breakdown
- **Pie charts** - Cost distribution by category
- **Line charts** - Cost trends over time

## Filtering and Analysis

### Date Range

Select your analysis period:
- Last 7 days
- Last 30 days
- Last 90 days
- Custom range

### Grouping Options

Group costs by:
- **Provider** - AWS, GCP, OpenAI, etc.
- **Service** - Compute, Storage, API calls
- **Department** - Based on hierarchy assignments
- **Project** - Based on hierarchy assignments
- **Team** - Based on hierarchy assignments

### Export Options

Export your analytics data:
- **CSV** - Spreadsheet format
- **PDF** - Report format

## Best Practices

1. **Regular reviews** - Check analytics weekly for trends
2. **Set budgets** - Use data to inform budget allocation
3. **Identify anomalies** - Look for unexpected spikes
4. **Compare periods** - Use historical data for planning
