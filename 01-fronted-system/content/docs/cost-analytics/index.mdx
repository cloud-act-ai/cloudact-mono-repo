## Filtering & Grouping

### Date Range Selection

| Option | Description |
|--------|-------------|
| **Today** | Current day (partial) |
| **Yesterday** | Previous complete day |
| **Last 7 days** | Rolling week |
| **Last 30 days** | Rolling month |
| **This month** | Current calendar month |
| **Last month** | Previous calendar month |
| **This quarter** | Current calendar quarter |
| **Last quarter** | Previous calendar quarter |
| **This year** | Current calendar year |
| **Custom** | Select start and end dates |

### Grouping Options

Group costs by multiple dimensions:

| Dimension | Description |
|-----------|-------------|
| **Provider** | AWS, GCP, Azure, OpenAI, etc. |
| **Service** | EC2, BigQuery, GPT-4, Slack, etc. |
| **Category** | Compute, Storage, AI, SaaS |
| **Region** | Geographic region |
| **Department** | Organizational department |
| **Project** | Assigned project |
| **Team** | Assigned team |
| **Tags/Labels** | Cloud resource tags |

### Applying Filters

1. Click **Filters** in the analytics toolbar
2. Select dimension (Provider, Service, etc.)
3. Choose values to include/exclude
4. Click **Apply**
5. Filters persist until cleared

## Comparison & Trends

### Period Comparison

Compare costs across time periods:

| Comparison | Shows |
|------------|-------|
| **vs Previous Period** | This week vs last week |
| **vs Same Period Last Month** | This week vs same week last month |
| **vs Same Period Last Year** | This month vs same month last year |

### Trend Indicators

| Indicator | Meaning |
|-----------|---------|
| **↑ Red** | Costs increased (negative) |
| **↓ Green** | Costs decreased (positive) |
| **→ Gray** | No significant change |

### Forecasting

Based on historical patterns, CloudAct projects:

- **End of Month** - Projected monthly total
- **End of Quarter** - Projected quarterly total
- **End of Year** - Projected annual total
- **Confidence Range** - Upper and lower bounds

<Callout type="info">
  Forecasts improve with more historical data. Allow 30+ days of data for accurate projections.
</Callout>

## Cost Allocation

### Hierarchy Integration

Costs flow through your organizational hierarchy:

```
Organization
└── Department (Engineering, Marketing, Sales)
    └── Project (Platform, Website, Mobile App)
        └── Team (Backend, Frontend, DevOps)
```

### Allocation Rules

| Source | Allocation |
|--------|------------|
| **Cloud resources** | By tags/labels |
| **GenAI usage** | By API key or project |
| **Subscriptions** | By explicit assignment |

### Unallocated Costs

Costs without hierarchy assignment:
- Shown as "Unassigned"
- Can be manually allocated
- Highlighted for attention

## Anomaly Detection

### Automatic Detection

CloudAct identifies unusual spending patterns:

| Anomaly Type | Description |
|--------------|-------------|
| **Spike** | Sudden increase above normal |
| **Sustained Increase** | Gradual but consistent rise |
| **New Service** | Costs from previously unused service |
| **Unusual Pattern** | Deviation from typical usage |

### Anomaly Alerts

When anomalies are detected:
- Highlighted in dashboard
- Optional notification sent
- Investigation notes can be added

### Setting Thresholds

Configure anomaly sensitivity:

1. Go to **Notifications**
2. Select **Anomaly Detection**
3. Set threshold (e.g., 20% above average)
4. Choose notification method

## Frequently Asked Questions

**Q: How often is cost data updated?**
A: Daily for most providers. Cloud data has 24-48 hour delay from providers.

**Q: Can I drill down to individual resources?**
A: Yes, for cloud costs. Click any service to see resource-level details.

**Q: How accurate are forecasts?**
A: Forecasts are based on historical trends. Accuracy improves with more data.

**Q: Can I create custom dashboards?**
A: Custom views coming soon. Use export for custom analysis.

**Q: How far back is historical data available?**
A: Typically 3 months of backfill, then ongoing from connection date.
