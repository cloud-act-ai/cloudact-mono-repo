Track and analyze your organization's SaaS subscription costs.

## Subscription Tracking

CloudAct tracks costs for all your SaaS tools and subscriptions:

| Category | Examples |
|----------|----------|
| **Productivity** | Slack, Microsoft 365, Google Workspace |
| **Development** | GitHub, GitLab, Jira, Linear |
| **Design** | Figma, Adobe Creative Cloud |
| **Marketing** | HubSpot, Mailchimp, Salesforce |
| **Security** | Okta, 1Password, Snyk |
| **Infrastructure** | Datadog, PagerDuty, Cloudflare |

## Cost Metrics

### Subscription Overview

| Metric | Description |
|--------|-------------|
| **Total Subscriptions** | Number of active subscriptions |
| **Monthly Spend** | Total monthly subscription cost |
| **Annual Spend** | Projected annual cost |
| **Per-Seat Cost** | Average cost per user |

### License Utilization

Track actual usage vs. purchased licenses:
- Active users vs. total licenses
- Utilization percentage
- Unused license cost

## Analytics Views

### By Category

Group subscriptions by:
- Tool category
- Department
- Project
- Team (via hierarchy)

### By Billing Cycle

| Cycle | Details |
|-------|---------|
| **Monthly** | Subscriptions billed monthly |
| **Annual** | Subscriptions billed yearly |
| **Usage-based** | Pay-as-you-go subscriptions |

### Cost Allocation

Assign subscriptions to your organizational hierarchy:
- Department
- Project
- Team

This enables cost center reporting and chargeback.

</Callout>

## Cost Optimization

### Identifying Savings

- **Unused licenses** - Reduce seat count
- **Duplicate tools** - Consolidate similar services
- **Plan downgrades** - Move to appropriate tier
- **Annual vs. monthly** - Save with annual billing

### Negotiation Insights

Use spending data for vendor negotiations:
- Historical spend trends
- Usage patterns
- Multi-year commitment options
