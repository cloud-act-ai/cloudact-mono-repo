## FOCUS Standard

CloudAct uses the **FOCUS 1.3** standard for cost normalization:

<Callout type="info">
  The FinOps Open Cost and Usage Specification (FOCUS) provides a unified schema for cloud billing data across providers.
</Callout>

Benefits:
- Consistent cost comparison across providers
- Standardized resource categorization
- Unified reporting format

## Cost Optimization

### Recommendations

CloudAct identifies optimization opportunities:
- **Rightsizing** - Underutilized resources
- **Reserved capacity** - On-demand to reserved savings
- **Idle resources** - Unused resources to terminate
- **Storage tiering** - Move data to cheaper storage classes

### Savings Tracking

Monitor your optimization progress:
- Realized savings
- Potential savings identified
- Savings plan utilization
