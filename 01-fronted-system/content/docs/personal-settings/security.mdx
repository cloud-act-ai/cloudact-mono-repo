Protect your CloudAct account with strong security practices.

## Accessing Security Settings

1. Click your **name/avatar** in the sidebar footer
2. Select the **Security** tab

![Personal Settings - Security Tab](/docs/images/profile-security.png)

## Password Management

### Resetting Your Password

1. Click **Reset Password**
2. Check your email for a reset link
3. Click the link (valid for 1 hour)
4. Enter your new password
5. Confirm and save

### Password Requirements

Your password must:
- Be at least 8 characters
- Include a mix of letters and numbers (recommended)
- Not be a commonly used password

<Callout type="tip">
  Use a password manager to generate and store strong, unique passwords.
</Callout>

## Security Best Practices

### Strong Passwords

| Do | Don't |
|----|-------|
| Use 12+ characters | Use personal info (birthdays, names) |
| Mix letters, numbers, symbols | Reuse passwords across sites |
| Use random phrases | Use common words or patterns |
| Use a password manager | Share passwords |

### Account Protection

1. **Unique password** - Don't reuse from other services
2. **Regular updates** - Change password periodically
3. **Secure email** - Protect your login email
4. **Log out** - Sign out on shared devices

### Recognizing Threats

Watch for:
- Unexpected password reset emails
- Unusual account activity
- Login from unknown locations
- Requests for your password

<Callout type="error">
  CloudAct will **never** ask for your password via email or phone.
</Callout>

## Session Management

### Active Sessions

Your account may have multiple active sessions:
- Web browser sessions
- API access tokens

### Signing Out

To end all sessions:
1. Click **Sign Out** in the sidebar footer
2. You'll be logged out of the current session

### Securing Shared Devices

When using shared or public computers:
1. Use incognito/private browsing
2. Never save password in browser
3. Always sign out when done
4. Clear browser history

Enhanced security with 2FA:

## Suspicious Activity

### What to Do

If you notice suspicious activity:

1. **Change password immediately**
2. **Check recent activity** - Review login history
3. **Contact support** - Report the incident
4. **Review integrations** - Revoke unknown access

### Reporting Issues

Contact security@cloudact.ai for:
- Suspected account compromise
- Security vulnerabilities
- Phishing attempts

## Account Recovery

### Lost Access

If you can't log in:

1. **Try password reset** - Use "Forgot Password"
2. **Check email access** - Ensure you can receive emails
3. **Contact support** - Last resort with identity verification

### Email Change

Your email cannot be changed directly. Contact support with:
- Current email address
- New email address
- Verification of identity
