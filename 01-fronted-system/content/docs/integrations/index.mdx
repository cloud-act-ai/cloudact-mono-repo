## How Integrations Work

```
1. Connect → 2. Authenticate → 3. Sync Data → 4. View Analytics
```

1. **Connect** - Select provider and start setup
2. **Authenticate** - Enter credentials (API key, service account, etc.)
3. **Sync Data** - Pipelines fetch your usage and billing data
4. **View Analytics** - See costs in your dashboard

## Security

Your credentials are protected:

| Protection | Description |
|------------|-------------|
| **Encrypted** | All credentials encrypted at rest |
| **Hidden** | Never displayed after saving |
| **Limited Access** | Only used for data fetching |
| **Audit Logged** | All access is logged |

<Callout type="tip">
  Create dedicated API keys with read-only permissions for CloudAct.
</Callout>

## Quick Setup

### Adding Your First Integration

1. Go to **Integrations** in the sidebar
2. Choose category (Cloud, GenAI, or Subscriptions)
3. Click **Connect** on the provider
4. Follow the setup wizard
5. Verify connection
6. Data appears in dashboard within minutes

## Managing Integrations

### Viewing Status

Each integration shows:
- Connection status (Connected, Error, Syncing)
- Last successful sync time
- Available data range

### Troubleshooting

Common issues:
- **Invalid credentials** - Re-enter or regenerate
- **Missing permissions** - Check required roles/scopes
- **No data** - Allow 24-48 hours for initial sync

## Best Practices

1. **Use dedicated credentials** - Don't share with other services
2. **Minimal permissions** - Only grant read access
3. **Regular review** - Check connections monthly
4. **Remove unused** - Delete integrations you no longer use
