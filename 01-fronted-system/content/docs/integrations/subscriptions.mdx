Add and manage your SaaS subscriptions for comprehensive cost tracking.

## Adding Subscriptions

### From Templates

CloudAct includes templates for popular services:

| Category | Examples |
|----------|----------|
| **Productivity** | Slack, Microsoft 365, Google Workspace, Notion |
| **Development** | GitHub, GitLab, Jira, Linear, Vercel |
| **Design** | Figma, Adobe Creative Cloud, Canva |
| **Marketing** | HubSpot, Mailchimp, Intercom |
| **Security** | Okta, 1Password, Snyk |
| **Infrastructure** | Datadog, PagerDuty, Cloudflare |

### Adding from Template

1. Go to **Integrations** → **Subscriptions**
2. Click **Add Subscription**
3. Search for the service
4. Select from templates
5. Enter your plan details
6. Click **Save**

### Custom Subscription

For services not in templates:

1. Click **Add Custom**
2. Enter subscription details:
   - Provider name
   - Plan name
   - Billing cycle
   - Cost
   - Start date
3. Click **Save**

## Subscription Details

### Required Information

| Field | Description |
|-------|-------------|
| **Provider** | Service name (e.g., Slack) |
| **Plan Name** | Your plan tier |
| **Billing Cycle** | Monthly, Annual, or Custom |
| **Cost** | Price per billing cycle |
| **Currency** | Payment currency |
| **Start Date** | When subscription started |

### Optional Information

| Field | Description |
|-------|-------------|
| **End Date** | Contract end date |
| **Seats/Licenses** | Number of user licenses |
| **Auto-Renew** | Whether it auto-renews |
| **Hierarchy** | Department/Project/Team assignment |
| **Notes** | Additional information |

## Cost Calculation

### How Costs Are Calculated

| Billing Cycle | Daily Cost |
|---------------|------------|
| Monthly | Cost ÷ Days in month |
| Annual | Cost ÷ 365 |
| Custom | Cost ÷ Days in period |

### Per-Seat Pricing

For subscriptions with per-user pricing:
- Enter per-seat cost
- Enter number of seats
- Total = Per-seat × Seats

## Hierarchy Assignment

Assign subscriptions to your org structure:

### Assigning to Hierarchy

1. Edit the subscription
2. Select **Department** (optional)
3. Select **Project** (optional)
4. Select **Team** (optional)
5. Save changes

### Cost Allocation

Assigned subscriptions appear in:
- Department cost reports
- Project cost reports
- Team cost reports

<Callout type="tip">
  Assign subscriptions to the lowest appropriate level for accurate cost allocation.
</Callout>

## Managing Subscriptions

### Editing Subscriptions

1. Click **Edit** on the subscription
2. Update details
3. Click **Save**

Changes apply to future cost calculations.

### Plan Changes

When your plan changes:

1. Edit the subscription
2. Update plan name and cost
3. Set the effective date
4. Save as a new version

Historical costs use the pricing active at that time.

### Canceling Subscriptions

When you cancel a subscription:

1. Edit the subscription
2. Set an **End Date**
3. Save changes

The subscription stops appearing in future costs.

### Deleting Subscriptions

To remove entirely:

1. Click **Delete** on the subscription
2. Confirm deletion
3. Historical data is removed

<Callout type="warning">
  Deleting removes all historical cost data. Use End Date to preserve history.
</Callout>
