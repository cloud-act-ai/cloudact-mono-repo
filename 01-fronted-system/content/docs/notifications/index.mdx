Manage your notification settings to stay informed about important events and cost alerts.

## Accessing Notifications

Click **Notifications** in the sidebar to view and configure your alerts.

## Notification Types

### Cost Alerts

Get notified when spending exceeds thresholds:

| Alert Type | Description |
|------------|-------------|
| **Daily Threshold** | When daily spend exceeds limit |
| **Weekly Threshold** | When weekly spend exceeds limit |
| **Monthly Budget** | When monthly budget is reached |
| **Anomaly Detection** | Unusual spending patterns detected |

### Pipeline Notifications

Stay informed about data processing:

| Alert Type | Description |
|------------|-------------|
| **Pipeline Failed** | When a pipeline encounters an error |
| **Pipeline Success** | Confirmation of successful runs |
| **Data Delay** | When expected data is delayed |

### Team Notifications

Updates about your organization:

| Alert Type | Description |
|------------|-------------|
| **Member Joined** | New team member added |
| **Member Left** | Team member removed |
| **Role Changed** | Member permissions updated |
| **Invite Accepted** | Pending invite was accepted |

### Billing Notifications

Payment and subscription updates:

| Alert Type | Description |
|------------|-------------|
| **Payment Due** | Upcoming payment reminder |
| **Payment Failed** | Payment processing issue |
| **Plan Changed** | Subscription plan updated |
| **Trial Ending** | Trial period expiring soon |

## Setting Up Alerts

### Cost Threshold Alert

1. Go to **Notifications**
2. Click **Add Alert**
3. Select **Cost Threshold**
4. Configure:
   - Threshold amount
   - Time period (daily/weekly/monthly)
   - Cost category (all, cloud, genai, subscriptions)
5. Choose notification channels
6. Click **Save**

### Anomaly Detection

Enable automatic detection of unusual spending:

1. Go to **Notifications**
2. Enable **Anomaly Detection**
3. Set sensitivity level:
   - **Low** - Only major anomalies
   - **Medium** - Moderate deviations
   - **High** - Minor variations

## Best Practices

1. **Set realistic thresholds** - Start higher and adjust based on patterns
2. **Enable anomaly detection** - Catch unexpected spending early
3. **Review regularly** - Check notification settings monthly
4. **Don't over-alert** - Too many notifications leads to alert fatigue

<Callout type="tip">
  Start with weekly budget alerts and add daily alerts for critical services.
</Callout>
