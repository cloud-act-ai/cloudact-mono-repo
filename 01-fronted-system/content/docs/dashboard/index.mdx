The Dashboard (Account Summary) provides a high-level overview of your organization's cloud and AI spending.

## Accessing the Dashboard

Click **Account Summary** → **Dashboard** in the sidebar to view your organization overview.

## Dashboard Overview

The dashboard displays:

### Key Metrics

| Metric | Description |
|--------|-------------|
| **Total Costs** | Combined spending across all integrations |
| **Cloud Costs** | AWS, GCP, Azure, OCI spending |
| **GenAI Costs** | OpenAI, Anthropic, Google AI spending |
| **Subscription Costs** | SaaS subscription spending |

### Cost Trends

- **Daily/Weekly/Monthly** views of spending
- **Period-over-period** comparison
- **Forecasted** spending projections

### Quick Actions

From the dashboard you can:

1. **View detailed analytics** - Drill down into specific cost categories
2. **Check pipeline status** - See recent pipeline runs
3. **Manage integrations** - Connect new providers

## Dashboard Cards

### Cloud Cost Summary

Shows breakdown by provider:
- AWS
- Google Cloud (GCP)
- Microsoft Azure
- Oracle Cloud (OCI)

### GenAI Cost Summary

Shows breakdown by provider:
- OpenAI
- Anthropic
- Google AI (Gemini)

### Recent Pipeline Runs

Lists the most recent pipeline executions with status:
- **Success** - Pipeline completed successfully
- **Running** - Pipeline currently executing
- **Failed** - Pipeline encountered an error

## Customizing Your View

### Date Range Selection

Choose from:
- Last 7 days
- Last 30 days
- Current month
- Custom date range

### Currency Display

All costs are displayed in your organization's configured currency (set in Org Settings → Organization → Locale Settings).

## Best Practices

1. **Check dashboard daily** - Monitor for unexpected spending spikes
2. **Set up alerts** - Configure notifications for cost thresholds
3. **Review trends** - Use historical data to forecast budgets
